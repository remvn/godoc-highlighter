# Suggested Commands

## Development
- `npm run dev`: Starts Vite in development mode with HMR for the extension.
- Chrome Extension Loading: Load the `dist` folder as an unpacked extension in `chrome://extensions`.

## Build & Release
- `npm run build`: Runs TypeScript compiler (`tsc`) and then builds the project using Vite.
- Output: The extension bundle is generated in the `dist` directory.

## System Utilities (Windows)
- `dir`: List files and directories.
- `findstr`: Search for strings in files (Windows equivalent of `grep`).
- `copy` / `move` / `del`: Basic file operations.
- `powershell`: For more advanced scripting if needed.
